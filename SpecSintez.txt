Что делать:

    Сменить SESSION_SECRET и пароль администратора

    Переписать историю Git, чтобы удалить .env из всех коммитов:

      "git rm --cached --ignore-unmatch .env" \

B. Не установлена библиотека chai для тестов 🔴

Во всех тестовых файлах используется require('chai'), но пакет отсутствует в package.json.

Решение:
bash

npm install --save-dev chai

C. Отсутствуют таблицы directions и product_directions в schema.sql 🔴

Файл schema.sql не содержит создания этих таблиц, хотя они используются в API и админке. При развертывании с нуля возникнут ошибки.

Решение: Добавить в schema.sql и test/setup.js:
sql

CREATE TABLE public.directions (
    id SERIAL PRIMARY KEY,
    name VARCHAR(255) NOT NULL,
    slug VARCHAR(255) UNIQUE NOT NULL,
    image_url VARCHAR(500),
    sort_order INTEGER DEFAULT 0
);

CREATE TABLE public.product_directions (
    product_id INTEGER REFERENCES products(id) ON DELETE CASCADE,
    direction_id INTEGER REFERENCES directions(id) ON DELETE CASCADE,
    PRIMARY KEY (product_id, direction_id)
);

D. XSS-уязвимость в отображении контента 🔴

В файлах detail-news.js, catalog.js, main-page.js HTML-контент (поля content, properties) вставляется без очистки:
javascript

// Опасно — выполнит любой скрипт
<div class="news-detail__content">${news.content}</div>

Решение: Установить и использовать DOMPurify:
bash

npm install dompurify jsdom

javascript

// Серверная очистка (в роутах)
const createDOMPurify = require('dompurify');
const { JSDOM } = require('jsdom');
const window = new JSDOM('').window;
const DOMPurify = createDOMPurify(window);

// Перед сохранением в БД
const cleanContent = DOMPurify.sanitize(req.body.content);

D. ORDER BY RANDOM() в API 🟡

Запрос ORDER BY RANDOM() для случайных товаров — медленный на больших таблицах.

Решение (эффективное):
javascript

// Получить максимальный ID активного товара
const maxId = await pool.query('SELECT MAX(id) FROM products WHERE is_active = true');
// Сгенерировать случайные ID и запросить по ним
const randomIds = new Set();
while (randomIds.size < limit) {
  randomIds.add(Math.floor(Math.random() * maxId) + 1);
}
const result = await pool.query(
  'SELECT * FROM products WHERE id = ANY($1) AND is_active = true LIMIT $2',
  [Array.from(randomIds), limit]
);
