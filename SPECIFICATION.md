## 1. Общее описание

Корпоративный сайт НПО «СпецСинтез». Включает публичную часть (каталог, новости, страницы) и административную панель для управления контентом.

---

## 2. Страницы сайта

| Страница | Файл | Описание |
|----------|------|----------|
| Главная | `index.html` | Hero с видео, отрасли, о компании, товары, преимущества, новости, CTA |
| Каталог | `catalog.html` | Аккордеон категорий, фильтр по направлениям, поиск, модальные окна |
| О компании | `about.html` | История, продукты, преимущества |
| Новости | `news.html` | Hero-новость + список |
| Новость | `news-detail.html` | Полный текст (загружается через API) |
| Производство ПАВ | `production.html` | Список продукции с PDF |
| Контрактное производство | `contract.html` | Информация + форма заявки |
| Партнёрам | `partners.html` | Схема работы, сотрудничество |
| Политика | `privacy.html` | Конфиденциальность и обработка данных |

---

## 3. База данных

### Таблицы

| Таблица | Назначение |
|---------|------------|
| `categories` | Иерархия категорий товаров |
| `directions` | Направления деятельности |
| `products` | Товары (название, slug, описание, свойства) |
| `product_categories` | Связь товаров с категориями (many-to-many) |
| `product_directions` | Связь товаров с направлениями (many-to-many) |
| `product_images` | Галерея изображений товара |
| `product_package_types` | Типы упаковок с объёмом |
| `package_types` | Справочник упаковок |
| `news` | Новости (заголовок, slug, контент, hero) |

---

## 4. API (публичные эндпоинты)

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/api/news` | Список новостей |
| GET | `/api/news/:slug` | Детальная новости |
| GET | `/api/categories` | Категории |
| GET | `/api/directions` | Направления |
| GET | `/api/products` | Товары (фильтры: category_id, direction_id, search) |
| GET | `/api/products-by-slug/:slug` | Полные данные товара |
| GET | `/api/random-products` | Случайные товары |
| POST | `/api/send-form` | Отправка формы |

---

## 5. Безопасность

| Механизм | Где применяется |
|----------|-----------------|
| CSRF-токены | Все формы в админке |
| XSS-защита | DOMPurify для TinyMCE |
| Content-Security-Policy | Helmet |
| Хеширование пароля | bcrypt для входа в админку |
| Валидация ввода | Проверка телефона, обязательных полей |

---

## 6. Кеширование

- In-memory кеш (60-600 секунд)
- HTTP-заголовки ETag + Cache-Control
- Полный сброс кеша при любых изменениях в админке
